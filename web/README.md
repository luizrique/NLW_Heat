# 📄 Sobre

✔ Projeto elabora através do evento <b>NLW Heat </b> da RocketSeat, onde foi desenvolvido o projeto do front-end para web.

---

# 💻 Instalação

```bash

# clonar diretório
git clone

# entrar no diretório
cd web

# instalar as dependências
yarn install

# roda o projeto
yarn dev
```
